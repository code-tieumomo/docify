# \[LMSX] Phần 2: Một số API cần thiết trong quá trình xây dựng ứng dụng

## Authentication

### Login

```bash
curl 'https://www.googleapis.com/identitytoolkit/v3/relyingparty/verifyPassword?key=AIzaSyAh2Au-mk5ci-hN83RUBqj1fsAmCMdvJx4' \
  -H 'accept: */*' \
  -H 'accept-language: en-US,en;q=0.9,vi;q=0.8' \
  -H 'content-type: application/json' \
  -H 'origin: https://base.mindx.edu.vn' \
  -H 'priority: u=1, i' \
  -H 'referer: https://base.mindx.edu.vn/' \
  -H 'sec-ch-ua: "Not/A)Brand";v="8", "Chromium";v="126", "Microsoft Edge";v="126"' \
  -H 'sec-ch-ua-mobile: ?0' \
  -H 'sec-ch-ua-platform: "Linux"' \
  -H 'sec-fetch-dest: empty' \
  -H 'sec-fetch-mode: cors' \
  -H 'sec-fetch-site: cross-site' \
  -H 'user-agent: Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/126.0.0.0 Safari/537.36 Edg/126.0.0.0' \
  -H 'x-client-version: Chrome/JsCore/7.20.0/FirebaseCore-web' \
  --data-raw '{"email":"<your_email>","password":"<your_password>","returnSecureToken":true}'
```

API này sẽ trả về một `idToken` và `teacherId` để xác thực người dùng. Sau đó chúng ta sẽ sử dụng `idToken`
và `teacherId` để xác thực trong các API khác.
